# **Day 1 Notes**

### Quick Tips
- ` "~" ` = home directory
- ` "/" ` = root (and path seperator)
- ` "." ` = current directory
- ` ".." ` = parent directory
- `ctrl + a` = places cursor at the beginning of the code line
- `ctrl + e` = places cursor at the end of the code line
